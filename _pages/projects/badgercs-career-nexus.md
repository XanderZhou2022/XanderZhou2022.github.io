## BadgerCS Career Nexus

<img src="/assets/img/projects/badgercs-career-nexus.jpg" alt="BadgerCS Career Nexus visual background" class="img-fluid rounded z-depth-1 mb-4">

BadgerCS Career Nexus is a web development course project for UW-Madison CS students. It provides a student-focused career hub for job discovery, job-detail browsing, community discussion, and dashboard prototyping.

### What It Does

- Provides filterable internship and job listings.
- Shows detailed job pages with company, salary, skills, and application-tracking interactions.
- Includes a community forum for posts, comments, and career discussion.
- Prototypes login, registration, profile, saved jobs, and user dashboard flows.

### My Work

- Contributed to a modular HTML/CSS/JavaScript codebase.
- Worked within a Git collaboration workflow with feature branches and pull requests.
- Helped shape the responsive user experience and student-oriented career platform concept.

### Tech Stack

HTML5, CSS3, Bootstrap 5, JavaScript, JSON-based mock data, GitHub Pages.
