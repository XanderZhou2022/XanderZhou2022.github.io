## Elasticsearch and Madison Interactive Kibana Map

<img src="/assets/img/projects/cs639-elasticsearch-map.png" alt="Madison interactive Kibana map" class="img-fluid rounded z-depth-1 mb-4">

This CS639 project focused on unstructured and geospatial data management with Elasticsearch and Kibana. The project integrated Madison-related data from multiple sources and used Elasticsearch queries to answer search, aggregation, and geospatial trivia questions.

### What It Does

- Indexed JSON, text, news, Wikipedia, Google Maps-style location data, weather/event data, and GeoJSON-style geographic records.
- Built Elasticsearch mappings and bulk ingestion workflows.
- Implemented full-text search, fuzzy search, phrase queries, highlighting, boosting, aggregation, and filtering.
- Visualized Madison places and enriched geospatial data with Kibana Maps.

### Tech Stack

Elasticsearch, Kibana, Python Elasticsearch client, Docker, Jupyter Notebook, GeoJSON.
